# ecommerce-sales-analysis_data_analytics
Performed data cleaning and preprocessing using Python (Pandas) and Excel and built an interactive dashboard in Power BI to analyze revenue, profit, and customer trends. Generated business insights on regional performance, category profitability, and marketing effectiveness using visual analytics.

# Repository Structure
analysis.py: Python script using Pandas for data cleaning, handling null values, and feature engineering (e.g., calculating Profit Margin).

ecommerce_sales_data.csv: The raw dataset used for the analysis.

cleaned_data.csv: The final processed dataset ready for visualization.

ecommerce_sales_dashboard.pbix: Interactive Power BI dashboard file.

# Tech Stack
Language: Python 3.x

Libraries: Pandas, NumPy

Visualization: Power BI Desktop

Data Format: CSV, Excel

# Key Insights
Regional Performance: Identified which regions contribute the most to the total revenue.

Product Trends: Analyzed which categories (e.g., Electronics, Fashion) have the highest profit-to-ad-spend ratio.

Monthly Growth: Tracked seasonal sales fluctuations across different years.
